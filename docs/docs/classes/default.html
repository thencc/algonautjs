connectAlgoWallet

  • connectAlgoWallet(clientListener?: WalletConnectListener): Promise<void>
    • Defined in index.ts:1723

    Connects to algo wallet via WalletConnect, calling subscribeToEvents. Implementation borrowed from Algorand Docs

    remarks

    There are three listeners you can use, defined by {@link WalletConnectListener}:

    • onConnect(payload: IInternalEvent) (payload.params[0] contains an array of account addresses)
    • onDisconnect()
    • onSessionUpdate(accounts: string[])
    example

    Usage:

    await algonaut.connectAlgoWallet({
      onConnect: (payload) => console.log('Accounts: ' + payload.params[0]),
      onDisconnect: () => console.log('Do something on disconnect'),
      onSessionUpdate: (accounts) => console.log('Accounts: ' + accounts)
    })

    We can use the onConnect listener to store an address in application state, for example, which allows us to conditionally display components depending on authentication status.

    Parameters

    • Optional clientListener: WalletConnectListener

      object of listener functions (see {@link WalletConnectListener})

    Returns Promise<void>
